🚀 Usage

  1. Preprocess Data: 
    python src/preprocessing.py
  2. Launch Dashboard: 
    streamlit run src/dashboard.py
  3. Open Dashboard (Local): http://localhost:8501
  4. Open Dashboard (Online): Streamlit Deployment

📊 Data Sources

Dataset Source Period
Temperature Records National Climate Center 1950–2023
Socioeconomic Data World Bank Open Data 1970–2023
Wildfire Incidents NASA FIRMS 2013–2023

📦 Dependencies

Install the required packages using:

pip install -r requirements.txt

🚨 Troubleshooting

🔧 Missing dependencies

pip install --upgrade -r requirements.txt

⚠️ Port conflict

streamlit run src/dashboard.py --server.port 8502

🧹 Clean & Reprocess

rm -rf data/processed/ && python src/preprocessing.py
